# 第 13 讲：前沿 CV 与生成模型：GAN、VAE、Diffusion

> 课程：Introduction to Artificial Intelligence · 复旦大学  
> 日期：2026-05-28 前后

## 本讲定位

本讲材料聚焦前沿 CV 生成模型：GAN 的极小极大博弈、WGAN 对 JS 散度问题的修正、VAE 的 ELBO、DDPM/Stable Diffusion 的逐步去噪，以及它们和视觉生成任务的关系。

## 学习目标

- 理解强化学习与监督学习的区别：数据来自智能体与环境交互。
- 掌握状态、动作、奖励、策略、价值函数、折扣回报和贝尔曼方程。
- 能把 Q-learning、REINFORCE、MCTS 与 CartPole 实验联系起来。

## 知识地图

| 模块 | 内容 |
| --- | --- |
| MDP | 用 S,A,P,R,γ 描述序列决策问题。 |
| 价值方法 | 估计 V(s) 或 Q(s,a)，再据此选择动作。 |
| 策略方法 | 直接优化 π(a|s)，适合连续动作和随机策略。 |
| 深度强化学习 | 用神经网络近似价值/策略，解决高维状态输入。 |

## 核心概念

- 探索与利用是 RL 核心矛盾：既要尝试新动作，也要利用已知高收益动作。
- 贝尔曼方程把长期回报拆成即时奖励和下一状态价值。
- CartPole 是理解状态离散化、奖励设计、训练曲线和算法对比的经典实验。

## 课堂讲解补充

强化学习把 AI 从静态预测推进到交互式决策。智能体通过试错获得奖励，并在探索与利用之间平衡。CartPole 项目让 MDP、Bellman 方程、Q-learning、策略梯度和 MCTS 不再停留在公式层面，而是落实到训练曲线、随机种子和算法对比。

### 复习组织方式

- **问题背景**：这节课为什么要引入这个概念。
- **方法主线**：算法或模型按什么步骤工作。
- **公式/代码**：至少抓住一个能落地计算的表达。
- **局限性**：说明它在哪些场景下会失效或需要改进。

## 公式与记忆点

- 回报：G_t=Σ_{k=0}∞ γ^k r_{t+k+1}。
- Bellman: Vπ(s)=Σ_a π(a|s)Σ_{s'}P(s'|s,a)[R+γVπ(s')]。
- Q-learning: Q(s,a)←Q(s,a)+α[r+γ max_a′Q(s′,a′)-Q(s,a)]。

## 易错点

- 奖励稀疏或设计不当会让智能体学不到有效策略。
- 训练曲线波动很正常，RL 需要多随机种子评估稳定性。
- 把环境交互数据当作独立同分布样本，会忽视 RL 的分布漂移。
