01 · 逻辑与知识表示：把世界写成可推理对象

符号主义 AI 的核心不是“记很多事实”，而是给事实一个可以组合、查询和推出新结论的形式。命题逻辑把陈述看成真假变量；谓词逻辑进一步引入对象、谓词和量词；知识图谱则把事实写成三元组。

(head entity, relation, tail entity) 例如 (Socrates, isA, Human)

命题逻辑

适合处理整体真假命题，例如 P→Q 与 P 可推出 Q。局限是不能表达“所有人”“某个人”“人与人之间关系”。

谓词逻辑

用 Human(x)、Mortal(x) 和 ∀x 表达结构化知识：∀x Human(x)→Mortal(x)。表达力更强，但推理代价也更高。

专业理解：知识图谱不是普通表格。普通表格强调存储，知识图谱强调关系路径和可推断性。FOIL 偏从例子归纳规则，PRA 偏用路径特征为关系推断打分。

02 · 搜索求解：先建模，再谈算法

搜索题的第一步永远不是写 BFS/A*，而是定义问题五要素：状态、动作、状态转移、路径代价、目标测试。若五要素含糊，算法复杂度和最优性都无从讨论。

算法展开依据适合场景关键性质
BFS深度从小到大边权相同、解较浅完备；单位代价下最优；内存开销大
DFS沿一条路径深入内存紧张、只需找到一个解不保证最优；可能陷入深分支
UCS累计代价 g(n)边权不同的最短路非负代价下最优
Greedy启发式 h(n)需要快但不强求最优可能被局部近似误导
A*f(n)=g(n)+h(n)最短路 + 有可靠启发h 可采纳且一致时最优
可采纳：0 ≤ h(n) ≤ h*(n)，即不高估真实剩余代价。

03 · 机器学习：预测函数、损失函数、优化器

机器学习的定义可压缩成三元组：经验 E、任务 T、性能指标 P。模型学习不是“程序员写规则”，而是从训练数据中调参数，使性能指标变好。

线性回归与梯度下降

ŷ = wᵀx + b
L = (1/n) Σᵢ (ŷᵢ-yᵢ)²
θ ← θ - η ∇θ L

线性模型的价值不在“简单”，而在它提供了现代神经网络的基本构件：线性层、损失函数、梯度更新、正则化和泛化误差。

SVM：最大间隔不是口号

二分类标签 y∈{−1,+1}，打分函数 f(x)=wᵀx+b。分类间隔：

z = y f(x)
z状态Hinge 损失 max(0,1-z)解释
2分对且间隔充足0远离边界，不再贡献损失
1正好在 margin 边界0刚好满足 SVM 约束
0.5分对但 margin 不足0.5正确但不够稳健
0在决策边界上1模型没有明确倾向
-0.5分错1.5错误并产生额外惩罚
-1分错且错得自信2错误置信度越强，损失越大
min_{w,b} 1/2 ||w||² + C Σᵢ max(0, 1-yᵢ(wᵀxᵢ+b))

考试判断：先算 z=yf(x)，再套 Hinge。不要把“分类正确”误等同于“损失为 0”；只有 z≥1 才是 0。

04 · 神经网络：非线性、反向传播与泛化

多层线性变换如果没有激活函数，本质仍等价于单层线性模型。因此神经网络的表达力来自“线性变换 + 非线性激活”的重复组合。

h = σ(W₁x+b₁), y = W₂h+b₂

反向传播

反向传播不是一种损失函数，而是用链式法则高效计算所有参数梯度的动态规划。核心是复用中间导数，避免对每个参数重新推导。

训练异常

训练损失和验证损失一起高：欠拟合。训练损失低但验证损失高：过拟合。梯度过小/过大：消失或爆炸，需要初始化、归一化、残差或梯度裁剪。

05 · CNN：图像结构带来的归纳偏置

把 1000×1000 图像直接拉平成向量接全连接层，会导致参数爆炸，也丢掉空间局部结构。CNN 的三个关键归纳偏置是局部连接、参数共享和平移等变性。

H_out = floor((H + 2P - K) / S) + 1
W_out = floor((W + 2P - K) / S) + 1
对象尺寸/参数常见错误
PyTorch 输入[N,C,H,W]把 channel 和 height 写反
卷积参数量C_out × (C_in × K_h × K_w + bias)忘记每个输出通道都有一个 filter
FLOPs约 2×C_out×C_in×K_h×K_w×H_out×W_out把参数量等同于计算量
感受野层数越深，单个特征看到的原图区域越大只看 kernel，不看 stride 累积

CNN 的“深刻理解”是：它不是万能视觉模型，而是把图像的局部性写进结构。对长距离关系、全局上下文和开放词表语义，后续需要注意力、检测/分割头或视觉语言模型补足。

06 · Transformer：从序列递归到全局交互

RNN 逐步读序列，天然保留顺序但难并行；Transformer 让所有 token 两两计算相关性，训练并行性强，但必须额外加入位置编码。

自注意力公式

Attention(Q,K,V)=softmax(QKᵀ / √d_k)V

典型尺寸题

设 n=5，B=2，d_model=12，h=3，则每头维度 d_k=d_v=4。

张量尺寸说明
输入 X2×5×12batch、序列长度、词向量维度
整体 Q/K/V2×5×12线性投影保持总维度
每头 Qᵢ/Kᵢ/Vᵢ2×5×412 维拆成 3 个 4 维子空间
Score = QKᵀ/√d_k2×5×5每个 query token 对所有 key token 打分
每头输出2×5×4softmax(score)V
拼接后2×5×12再接输出投影 W_O

位置编码

PE(pos,2i)=sin(pos/10000^{2i/d_model})
PE(pos,2i+1)=cos(pos/10000^{2i/d_model})

位置编码加在输入 embedding 上：X_pos=X+PE。尺寸是 n×d_model=5×12，与 batch 相加时广播为 2×5×12。它不是分别加到 Q、K、V 上。

参数量

若 W_Q/W_K/W_V/W_O 都是 12→12 且带偏置，每个线性层参数量为 12×12+12=156。Q/K/V 三层共 468，加输出投影共 624。

07 · CV/NLP 应用：检测、分割、生成

图像分类只回答“图里主要是什么”；目标检测回答“哪里有什么”；语义分割回答“每个像素属于什么类”；实例分割进一步区分同类不同个体。

任务输出指标核心难点
分类单个或多个类别Accuracy / F1类别不平衡会使 Accuracy 失真
检测boxes + labels + scoresIoU / AP / mAP小目标、遮挡、多尺度、框偏移
语义分割每像素类别mIoU边界精细度与上下文语义
实例分割每对象 maskmask AP同类实例粘连与遮挡
视频分割跨帧一致的对象 maskJ&F 等遮挡、消失再现、镜头切换
IoU = area(B_pred ∩ B_gt) / area(B_pred ∪ B_gt)

R-CNN 系列的主线是效率提升：R-CNN 对每个候选框单独跑 CNN；Fast R-CNN 共享整图特征；Faster R-CNN 用 RPN 学习候选区域。YOLO/SSD 则走 one-stage 路线，直接密集预测，速度更快。

NLP 部分与 Transformer 汇合：词向量把离散 token 映射到连续空间，语言模型通过上下文预测学习语义结构，多模态模型再把图像区域、文本 token 和生成目标对齐。
