第 5 讲：机器学习（二）：SVM 与模型选择评估

用最大间隔理解 SVM，并系统梳理模型选择、验证集、交叉验证与分类指标。

学习目标

  • 理解 SVM 的目标是寻找最大间隔超平面。
  • 知道软间隔、核函数如何处理噪声与非线性。
  • 能根据类别不平衡程度选择 Accuracy、F1、ROC-AUC 或 PR-AUC。

知识地图

模块内容
SVM凸优化、支持向量、最大间隔、软间隔。
模型选择验证集、K 折交叉验证、超参数搜索。
评估指标混淆矩阵、精确率、召回率、F1、ROC/PR 曲线。

核心概念讲解

要点 1

支持向量是距离决策边界最近、真正决定间隔的样本。

要点 2

核技巧允许在高维特征空间做线性分割，而不显式计算高维映射。

要点 3

类别极不平衡时 Accuracy 可能误导，应重点看召回、F1 和 PR-AUC。

课堂讲解补充

SVM 与模型评估帮助我们从“能训练模型”进入“能判断模型是否可靠”。最大间隔思想强调泛化边界，交叉验证强调调参规范，混淆矩阵和 AUC/PR-AUC 则要求我们根据真实任务风险选择指标。

复习时可以按“问题从哪里来、方法怎么做、公式或代码落在哪里、效果怎么评价、局限是什么”来整理。

与课程主线的连接

公式速查与解释

  • 硬间隔目标：min 1/2||w||², s.t. yᵢ(wᵀxᵢ+b) ≥ 1。
  • Precision=TP/(TP+FP)，Recall=TP/(TP+FN)。
  • F1=2PR/(P+R)。
阅读建议：不要孤立背公式；每个公式都要能说出变量含义、适用条件和在代码/实验中的位置。

从课堂到项目的标准流程

步骤操作要点
1. 定义任务先说清输入、输出、假设与评价指标，避免只罗列算法名。
2. 建立表示选择逻辑符号、状态空间、特征向量、张量或 MDP 状态。
3. 选择模型根据数据规模、结构先验和计算预算选择规则、搜索、线性模型、神经网络或 RL 方法。
4. 训练/推理明确损失函数、优化过程、推理步骤和失败模式。
5. 评估迭代用与任务匹配的指标验证效果，并记录可复现实验设置。

代码/伪代码示例

下面的代码只保留课堂最需要抓住的骨架；真正练习时再对照 notebooks、讲义和项目文件补全细节。

from sklearn.model_selection import GridSearchCV
from sklearn.svm import SVC

param_grid = {"C": [0.1, 1, 10], "gamma": ["scale", 0.01, 0.1]}
search = GridSearchCV(SVC(kernel="rbf"), param_grid, scoring="f1_macro", cv=5)
search.fit(X_train, y_train)
print(search.best_params_, search.best_score_)

常见误区

  • 只报告最高 Accuracy 但不说明数据分布，会掩盖少数类失败。
  • 在测试集上反复试超参数，本质上泄漏了测试信息。
考试与汇报提醒：回答问题时尽量使用“任务定义 → 方法原理 → 关键公式/代码 → 指标 → 局限”的结构。
