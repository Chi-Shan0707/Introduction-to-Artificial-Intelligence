# 第 6 讲：机器学习（三）：无监督学习

> 课程：Introduction to Artificial Intelligence · 复旦大学  
> 日期：2026-04-09  
> 资料：无 PDF；基于 lec6.md 与补充材料

## 本讲定位

从没有标签的数据中发现结构：聚类、降维与概率生成视角。

## 学习目标

- 理解无监督学习面向“结构发现”而不是直接预测标签。
- 掌握 K-Means 的迭代更新、PCA 的最大方差投影和 GMM 的软聚类。
- 能把聚类/降维用作后续监督学习或可视化的预处理。

## 知识地图

| 模块 | 内容 |
| --- | --- |
| K-Means | 指定簇数 K，交替分配样本和更新中心。 |
| PCA | 寻找最大方差方向，用低维坐标保留主要信息。 |
| GMM | 用多个高斯分布混合建模，EM 估计隐变量。 |

## 核心概念

- K-Means 假设簇近似球形，且需要预先设定 K。
- PCA 是线性降维，适合去相关、压缩和可视化。
- GMM 给出软分配概率，比 K-Means 更能表达不确定性。

## 课堂讲解补充

无监督学习的价值在于从没有标签的数据中发现结构。K-Means、PCA、GMM 分别代表硬聚类、线性降维和概率软聚类三种思路。它们常常不是最终模型，而是数据理解、可视化、预处理和异常发现的重要工具。

### 复习组织方式

- **问题背景**：这节课为什么要引入这个概念。
- **方法主线**：算法或模型按什么步骤工作。
- **公式/代码**：至少抓住一个能落地计算的表达。
- **局限性**：说明它在哪些场景下会失效或需要改进。

## 公式与记忆点

- K-Means 目标：min Σᵢ ||xᵢ-μ_{cᵢ}||²。
- PCA：选择协方差矩阵最大特征值对应方向。
- EM：E 步估计隐变量后验，M 步最大化期望完全数据似然。

## 易错点

- 聚类标签编号没有天然语义，0/1/2 只是算法输出的编号。
- 降维图看起来分开，不代表原空间一定线性可分。

## 课后巩固

- 解释为什么 K-Means 对初始化敏感。
- 用 PCA 将高维特征压到二维并可视化。
