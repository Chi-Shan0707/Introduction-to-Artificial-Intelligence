03

重点难点：考试和汇报最容易卡住的地方

  • K-Means 假设簇近似球形，且需要预先设定 K。
  • PCA 是线性降维，适合去相关、压缩和可视化。
  • GMM 给出软分配概率，比 K-Means 更能表达不确定性。
  • K 值、初始化和尺度会显著影响 K-Means。
  • PCA 是线性降维，二维图分开不代表原空间可线性分类。
  • GMM 比 K-Means 更柔和，但协方差估计对样本量和异常值敏感。

优化隐变量通过 EM 算法更新 GMM 责任度；通过 K-Means 更新硬分类标签；通过特征值分解计算主坐标。失败于局部最优解。
解释结构评估聚类紧凑度（轮廓系数）或降维的累积解释方差比（EVR）。分析低维主成分含义。失败于低维度投影丢失判别性信息。
用于可视化或下游任务将降维后的 Z ∈ R^{N×k} 输入到分类器中，或利用 TSNE/PCA 在 2D 空间绘制散点图。失败于过度降维导致下游分类精度雪崩。

公式推导：从目标到结论

下面不是孤立公式，而是推导链：先写目标，再说明约束，最后解释算法为什么长成现在这样。

K-Means：min_{c,μ} Σᵢ ||xᵢ-μ_{cᵢ}||²。
PCA：max_{||u||=1} uᵀΣu，解为协方差矩阵最大特征值对应特征向量。
GMM：p(x)=Σ_k π_k N(x|μ_k,Σ_k)。
  1. K-Means 固定中心时分配最近簇，固定分配时更新簇均值；两步交替下降目标函数。
  2. PCA 从最大化投影方差出发，拉格朗日推导得到 Σu=λu。
  3. GMM 把簇归属看成隐变量，E 步估计责任度，M 步更新混合权重、均值和协方差。

\n# -*- coding: utf-8 -*-
# 第 6 讲: 无监督学习 - NumPy 纯手工实现 K-Means 聚类与 PCA 降维
import numpy as np

# ================= K-Means 聚类 =================
def k_means(X, K, max_iter=20):
    # 1. 随机初始化 K 个中心点
    centroids = X[np.random.choice(X.shape[0], K, replace=False)]
    for _ in range(max_iter):
        # 2. 分配步骤：计算样本到每个中心点的欧氏距离，并归属到最近中心
        distances = np.linalg.norm(X[:, np.newaxis] - centroids, axis=2)
        labels = np.argmin(distances, axis=1)
        # 3. 更新步骤：中心点移动到对应簇样本的均值处
        new_centroids = np.array([X[labels == k].mean(axis=0) if len(X[labels == k]) > 0 else centroids[k] for k in range(K)])
        if np.allclose(centroids, new_centroids):
            break
        centroids = new_centroids
    return labels, centroids

# ================= PCA 主成分分析 =================
def pca(X, k_components):
    # 1. 中心化数据
    X_mean = np.mean(X, axis=0)
    Xc = X - X_mean
    # 2. 计算协方差矩阵 S = 1/N * Xc.T @ Xc
    S = np.dot(Xc.T, Xc) / len(X)
    # 3. 特征值分解
    eigvals, eigvecs = np.linalg.eigh(S)
    # 4. 按特征值降序排序，取前 k 个特征向量
    idx = np.argsort(eigvals)[::-1]
    W = eigvecs[:, idx[:k_components]]
    # 5. 投影到低维空间
    Z = np.dot(Xc, W)
    return Z, W

# 测试数据: 6个二维数据点
X_toy = np.array([[1.0, 2.0], [1.5, 1.8], [5.0, 8.0], [6.0, 9.0], [1.2, 1.5], [5.5, 8.5]])

# 执行 K-Means
labels, centroids = k_means(X_toy, K=2)
print("--- K-Means 结果 ---")
print(f"数据类别分配 labels: {labels}")
print(f"最终聚类中心 centroids:\n{centroids}")

# 执行 PCA (降为 1 维)
Z, W = pca(X_toy, k_components=1)
print("\n--- PCA 结果 ---")
print(f"投影后低维坐标 Z:\n{Z}")
print(f"主成分投影矩阵 W:\n{W}")

doubao.md
本次课程主要介绍了无监督学习中的聚类与降维方法，重点讲解了K-means、DBSCAN和PCA算法，并探讨了高斯混合模型（GMM）及其EM算法。
​​核心思想​​：无监督学习用于在没有标签的情况下，发现数据本身的结构和规律，例如数据的相似性、密度等。
​​目标​​：将数据划分为K个簇(cluster)，使得簇内数据点间的距离（如欧氏距离）尽可能小，而不同簇间的距离尽可能大。
​​过程​​：采用交替优化策略，随机初始化K个中心点，然后反复进行以下步骤：将每个数据点分配给最近的中心点，再根据被分配的点重新计算中心点位置，直至收敛。
​​缺点​​：需要预先指定K值；对异常值敏感；在处理复杂数据形态（如月牙形）时效果不佳；计算复杂度随数据量增大而急剧上升。
​​核心思想​​：基于密度的聚类，认为密度高的区域是核心点，会形成簇；密度低的区域则为噪声点。
​​关键参数​​：ε（邻域半径）和MinPts（邻域内至少需要的点数）。
​​过程​​：随机选择一个未访问的点，若其ε邻域内点数≥MinPts，则该点为核心点，并将其邻域内的所有点（包括核心点和边界点）加入该簇；否则为噪声点。
​​优点​​：无需预先指定簇的数量；能发现任意形状的簇；能有效识别噪声点。
​​缺点​​：对参数ε和MinPts的选择敏感；在不同尺度的簇上表现不佳。
​​核心思想​​：通过线性变换，找到数据中方差最大的方向（主成分），将高维数据投影到这些主成分上，从而实现降维，同时保留尽可能多的数据信息。
对协方差矩阵进行特征值分解，得到特征值和特征向量。
​​应用实例​​：人脸识别。通过PCA将人脸图像（高维像素数据）降维到少数几个主成分（特征脸）进行存储和比对，从而实现高效的人脸识别。
​​核心思想​​：将数据视为由多个高斯分布（正态分布）混合而成，每个高斯分布对应一个潜在的簇。
​​与K-means的区别​​：GMM提供的是​​概率性​​的归属结果，即每个样本属于各个簇的概率，而不是确定性的归属。这使得GMM在处理边界模糊或复杂分布的数据时更为鲁棒。
​​E-step (Expectation)​​：在模型参数已知的情况下，计算每个数据点属于各个高斯分布的概率。
​​M-step (Maximization)​​：在数据归属概率已知的情况下，更新模型参数（各高斯分布的均值、协方差矩阵和混合权重）。
​​过程​​：反复执行E-step和M-step，直至模型参数收敛。

## 0. 本讲主线
- **化繁为简**：把高维、复杂的数据结构压缩成更简单、更易处理的形式
- **无中生有**：用生成模型去解释数据从哪里来
- 无监督学习没有标签，目标是从特征中自动发现结构
这意味着无监督学习的关键不在“答案是什么”，而在于：
## 1. 监督学习 vs 无监督学习
### 1.1 监督学习
例如图像分类里，数据不仅有形状、颜色等特征，还有人工标注的类别。
### 1.2 无监督学习
这时模型无法直接“照答案分类”，只能依赖特征本身去寻找规律。
课堂里提到的一个核心观点是：**相似性不是天然存在的，而是由特征和距离共同定义的。**
- 图像分割时，可以根据颜色、纹理、亮度等特征判断像素是否属于同一区域
- 分子分类时，可以根据 atomic features 判断原子或分子是否属于同类
所以，无监督学习的第一步往往不是算法本身，而是“特征怎么选、距离怎么算”。
## 2. 无监督学习的两类基本任务
### 2.1 聚类（Clustering）
直观上，这就是课堂里说的“看形状、看颜色、看特征，把东西分开”。
### 2.2 降维（Dimensionality Reduction）
降维的目标是把高维数据映射到更低维空间，同时尽量保留数据的主要结构。
### 2.3 生成建模（Generative Modeling）
生成模型不只是“分组”，还要解释“数据是怎么生成出来的”。
高斯混合模型就是典型代表：它假设数据来自多个高斯分布的混合。
## 3. K-Means 聚类
K-Means 是最经典的聚类算法之一。它的核心思想非常直接：
### 3.1 优化目标
X = \{x_1, x_2, \dots, x_n\}, \quad x_i \in \mathbb{R}^d
- $z_i \in \{1,2,\dots,K\}$ 表示样本 $x_i$ 的簇编号
J = \sum_{i=1}^n \|x_i - \mu_{z_i}\|^2
J = \sum_{k=1}^K \sum_{i:z_i=k} \|x_i - \mu_k\|^2
这个目标的含义是：让每个样本都尽量靠近自己所属簇的中心。
### 3.2 交替优化
K-Means 不能一次性直接求出全局最优，所以采用交替优化：
#### 3.2.1 固定中心时，样本应该分给谁？
对每个样本 $x_i$，若中心固定，则应选择距离最近的中心：
z_i = \arg\min_{k} \|x_i - \mu_k\|^2
#### 3.2.2 固定分配时，中心应该放在哪里？
对第 $k$ 个簇，记其中样本数为 $n_k$，目标为：
J_k(\mu_k) = \sum_{i:z_i=k} \|x_i - \mu_k\|^2
\frac{\partial J_k}{\partial \mu_k}
= \sum_{i:z_i=k} 2(\mu_k - x_i)
= 2n_k \mu_k - 2\sum_{i:z_i=k} x_i
\mu_k = \frac{1}{n_k} \sum_{i:z_i=k} x_i

## 4. PCA：主成分分析
### 4.1 预处理：中心化
X = [x_1^T; x_2^T; \dots; x_n^T] \in \mathbb{R}^{n \times d}
\bar{x} = \frac{1}{n} \sum_{i=1}^n x_i
\widetilde{x}_i = x_i - \bar{x}
为什么要中心化？因为 PCA 关心的是“围绕均值的波动结构”，而不是整体平移。
### 4.2 单个主成分的推导：最大化投影方差
设我们要找一个单位向量 $w$，把数据投影到这条直线上：
y_i = w^T \tilde{x}_i, \quad \|w\|=1
= \frac{1}{n} \sum_{i=1}^n (w^T \tilde{x}_i)^2
= w^T \left(\frac{1}{n} \sum_{i=1}^n \tilde{x}_i \tilde{x}_i^T \right) w
S = \frac{1}{n} X_c^T X_c
这就是一个标准的 Rayleigh quotient 最大化问题。

无监督学习深度补充：没有标签时学习结构

无监督学习不是没有目标，而是目标不由人工标签直接给出。K-Means 假设样本可由若干中心代表，PCA 假设主要信息集中在高方差方向，GMM 假设数据由多个高斯分布混合生成，EM 在隐变量未知时交替估计分配和参数。

方法优化对象输出局限
K-Means类内平方距离簇中心与簇标签偏好球形簇，需指定 K
PCA最大方差/最小重构误差低维主成分只能捕捉线性结构
GMM数据似然软聚类概率对初始化敏感，可能局部最优
EM含隐变量似然下界E 步后验、M 步参数不是全局最优保证

深层联系：K-Means 可看作一种硬分配聚类；GMM 是软分配概率模型；EM 是处理隐变量优化的通用思想。PCA 则不是聚类器，而是表示压缩方法。降维不一定提升分类准确率，因为低方差方向也可能有高判别信息。
