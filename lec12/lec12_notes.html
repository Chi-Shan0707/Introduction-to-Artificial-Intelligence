02

知识地图

目标检测

在图中找出多个对象的位置和类别，输出 bounding boxes + labels + scores。

图像分割

语义分割给每个像素类别，实例分割进一步区分同类不同对象。

视频分割

在时间维度保持对象一致性，处理遮挡、消失再现和镜头变化。

生成与 NLP

扩散模型、文本表示和语言模型共同支撑图文生成与视觉语言模型。

03

核心概念

  • Two-stage 检测先产生候选区域再分类/回归，精度强但流程较重；One-stage 直接密集预测，速度更快。
  • R-CNN 引入 CNN 特征用于检测；Fast/Faster R-CNN 用共享特征和 RPN 提升效率。
  • YOLO 把检测转成一次前向预测，适合实时场景。
  • 分割任务比检测更细粒度，模型必须同时理解语义和空间边界。
04

课堂讲解补充

第 12 讲是课程中计算机视觉应用最密集的一讲：检测解决“在哪里有什么”，分割解决“每个像素属于什么/哪个实例”，视频分割进一步加入时间一致性，图像生成和 NLP 则把视觉推向多模态。CNN 在这里从分类骨干扩展为视觉系统的特征提取器。

复习口径
先说明问题背景，再讲方法主线，最后落到公式、代码或指标；这比背术语更接近课堂答题。
05

公式与记忆点

  • IoU = area(box_pred ∩ box_gt) / area(box_pred ∪ box_gt)。
  • mAP：在多个类别和 IoU 阈值下对 AP 求平均。
  • 语义分割常用 mIoU：对各类别 IoU 取平均。
06

代码锚点

def box_iou(a, b):
    ax1, ay1, ax2, ay2 = a
    bx1, by1, bx2, by2 = b
    inter = max(0, min(ax2, bx2)-max(ax1, bx1)) * max(0, min(ay2, by2)-max(ay1, by1))
    area_a = max(0, ax2-ax1) * max(0, ay2-ay1)
    area_b = max(0, bx2-bx1) * max(0, by2-by1)
    return inter / (area_a + area_b - inter + 1e-8)
07

易错点与课后巩固

易错点

  • 检测框看起来接近但 IoU 低，可能是小目标或框偏移造成。
  • 把语义分割和实例分割混淆：前者不区分同类个体，后者区分。
  • 只看可视化效果不看 mAP/mIoU，难以专业比较模型。

课后巩固

  • 为一张图片分别写出分类、检测、语义分割、实例分割的输出格式。
  • 解释 Faster R-CNN 中 RPN 的作用。
  • 用 IoU 函数计算两个预测框的重叠程度。
