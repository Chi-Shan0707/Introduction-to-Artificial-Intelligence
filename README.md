## 教学进度

| 周次 | 主题 | 内容 |
|------|------|------|
| 第1周 | 人工智能绪论 | 定义、起源、发展、未来展望 |
| 第2周 | 逻辑与知识表示 | 形式逻辑、数理逻辑、知识图谱、概率图 |
| 第3周 | 搜索求解 | 图遍历、无信息搜索、启发式搜索 |
| 第4周 | 机器学习（一） | 线性模型、线性回归、线性分类 |
| 第5周 | 机器学习（二） | 支持向量机、模型选择与评估 |
| 第6周 | 机器学习（三） | 无监督学习、K-Means、PCA、高斯混合模型 |
| 第7周 | 深度学习（一） | 概要、网络结构、梯度计算 |
| 第8周 | 深度学习（二） | CNN详解、CNN发展 |
| 第9周 | 深度学习（三） | CV（检测/分割/生成）、NLP基础 |
| 第10周 | 深度学习（四） | RNN、LSTM、Transformer |
| 第11周 | 深度学习（五） | LLM（BERT/GPT/MOSS）、多模态大模型 |
| 第12周 | 强化学习（一） | 简介、马尔可夫过程、贝尔曼方程 |
| 第13周 | 强化学习（二） | 基于价值/策略、深度强化学习 |
| 第14周 | 前沿选讲 | 图像生成、视频生成、具身智能、可信AI |
| 第15-16周 | 项目汇报 | 小组项目汇报 |
| 第17-18周 | 考试 | 线下考试 |
