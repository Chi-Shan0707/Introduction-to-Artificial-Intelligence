第 7 讲：深度学习（一）：神经网络基础

从线性分类器扩展到多层神经网络，理解反向传播、正则化与优化。

学习目标

  • 理解神经网络是“线性变换 + 非线性激活”的层级组合。
  • 掌握反向传播利用链式法则高效计算梯度。
  • 知道 Dropout、Weight Decay、BatchNorm、学习率调度等训练稳定化方法。

知识地图

模块内容
网络结构输入层、隐藏层、输出层；深度来自多层非线性组合。
反向传播前向计算损失，反向传播梯度，优化器更新参数。
训练技巧正则化防过拟合，优化策略提升收敛速度和稳定性。

核心概念讲解

要点 1

没有非线性激活，多层线性网络仍等价于单层线性模型。

要点 2

反向传播不是一个新损失，而是计算梯度的动态规划过程。

要点 3

训练深度网络时要同时关注欠拟合、过拟合、梯度消失/爆炸和数据归一化。

课堂讲解补充

神经网络基础是后续 CNN、Transformer 和深度强化学习的共同底座。反向传播让复杂网络可以端到端训练，正则化和优化技巧决定模型能否稳定泛化。学习本讲时要始终追踪张量形状、梯度流和训练/验证曲线。

复习时可以按“问题从哪里来、方法怎么做、公式或代码落在哪里、效果怎么评价、局限是什么”来整理。

公式速查与解释

  • 链式法则：∂L/∂x = ∂L/∂y · ∂y/∂x。
  • ReLU(x)=max(0,x)。
  • Weight Decay: L_total=L_data+λ||θ||²。
阅读建议：不要孤立背公式；每个公式都要能说出变量含义、适用条件和在代码/实验中的位置。

从课堂到项目的标准流程

步骤操作要点
1. 定义任务先说清输入、输出、假设与评价指标，避免只罗列算法名。
2. 建立表示选择逻辑符号、状态空间、特征向量、张量或 MDP 状态。
3. 选择模型根据数据规模、结构先验和计算预算选择规则、搜索、线性模型、神经网络或 RL 方法。
4. 训练/推理明确损失函数、优化过程、推理步骤和失败模式。
5. 评估迭代用与任务匹配的指标验证效果，并记录可复现实验设置。

代码/伪代码示例

下面的代码只保留课堂最需要抓住的骨架；真正练习时再对照 notebooks、讲义和项目文件补全细节。

import torch
from torch import nn

class MLP(nn.Module):
    def __init__(self, in_dim=784, hidden=256, num_classes=10):
        super().__init__()
        self.net = nn.Sequential(
            nn.Flatten(),
            nn.Linear(in_dim, hidden),
            nn.ReLU(),
            nn.Dropout(0.2),
            nn.Linear(hidden, num_classes),
        )

    def forward(self, x):
        return self.net(x)

model = MLP()
criterion = nn.CrossEntropyLoss()
optimizer = torch.optim.AdamW(model.parameters(), lr=1e-3, weight_decay=1e-4)

常见误区

  • 只增加层数不一定更好：数据量、正则化和优化都会限制效果。
  • 忘记 `optimizer.zero_grad()` 会导致梯度累积，训练行为异常。
考试与汇报提醒：回答问题时尽量使用“任务定义 → 方法原理 → 关键公式/代码 → 指标 → 局限”的结构。
