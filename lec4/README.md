# 第 4 讲：机器学习（一）：线性模型

> 课程：Introduction to Artificial Intelligence · 复旦大学  
> 日期：2026-03-26  
> 资料：人工智能-4-线性模型.pdf、本讲 Markdown/实践材料

## 本讲定位

从机器学习三要素出发，建立线性回归、线性分类、损失函数与梯度下降的基本框架。

## 学习目标

- 理解经验 E、任务 T、性能 P 如何组成机器学习定义。
- 能写出线性模型的预测函数、损失函数和参数更新。
- 把分类、回归、聚类、降维等学习任务放入统一框架。

## 知识地图

| 模块 | 内容 |
| --- | --- |
| 学习任务 | 分类、回归、聚类、降维、密度估计。 |
| 模型形式 | 线性函数加非线性输出；复杂模型也常以线性层为构件。 |
| 优化方法 | 用损失函数衡量误差，再用梯度下降寻找更优参数。 |

## 核心概念

- 线性回归最小化均方误差，适合连续值预测。
- Logistic 回归把线性打分通过 sigmoid/softmax 转为概率。
- 训练集、验证集、测试集承担不同职责，不能混用。

## 课堂讲解补充

线性模型是现代机器学习和深度学习的底层构件。即使后来使用 CNN、Transformer 或强化学习网络，最后也离不开线性层、损失函数、梯度和优化器。本讲的关键是把“预测函数—损失函数—参数更新”这一闭环彻底打通。

### 复习组织方式

- **问题背景**：这节课为什么要引入这个概念。
- **方法主线**：算法或模型按什么步骤工作。
- **公式/代码**：至少抓住一个能落地计算的表达。
- **局限性**：说明它在哪些场景下会失效或需要改进。

## 公式与记忆点

- 线性回归：ŷ = wᵀx + b。
- MSE: L = (1/n)Σ(ŷᵢ-yᵢ)²。
- 梯度下降：θ ← θ - η∇θL。

## 易错点

- 线性模型不等于简单：特征设计、正则化和评估决定实际效果。
- 测试集不能参与调参，否则性能估计会偏乐观。

## 课后巩固

- 手推一维线性回归的 MSE 梯度。
- 说明学习率过大与过小分别会发生什么。
