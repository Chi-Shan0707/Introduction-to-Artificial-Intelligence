Introduction to Artificial Intelligence

第 4 讲：机器学习（一）：线性模型

从机器学习三要素出发，建立线性回归、线性分类、损失函数与梯度下降的基本框架。

学习目标

  • 理解经验 E、任务 T、性能 P 如何组成机器学习定义。
  • 能写出线性模型的预测函数、损失函数和参数更新。
  • 把分类、回归、聚类、降维等学习任务放入统一框架。

知识地图

模块内容
学习任务分类、回归、聚类、降维、密度估计。
模型形式线性函数加非线性输出；复杂模型也常以线性层为构件。
优化方法用损失函数衡量误差，再用梯度下降寻找更优参数。

核心概念讲解

要点 1

线性回归最小化均方误差，适合连续值预测。

要点 2

Logistic 回归把线性打分通过 sigmoid/softmax 转为概率。

要点 3

训练集、验证集、测试集承担不同职责，不能混用。

课堂讲解补充

线性模型是现代机器学习和深度学习的底层构件。即使后来使用 CNN、Transformer 或强化学习网络，最后也离不开线性层、损失函数、梯度和优化器。本讲的关键是把“预测函数—损失函数—参数更新”这一闭环彻底打通。

表达结构

复习时可以按“问题从哪里来、方法怎么做、公式或代码落在哪里、效果怎么评价、局限是什么”来整理。

与课程主线的连接

复习时顺手把它和前后几讲连起来：这一讲解决了什么问题，又给后面的模型或实验留下了哪些工具。

公式速查与解释

  • 线性回归：ŷ = wᵀx + b。
  • MSE: L = (1/n)Σ(ŷᵢ-yᵢ)²。
  • 梯度下降：θ ← θ - η∇θL。
阅读建议：不要孤立背公式；每个公式都要能说出变量含义、适用条件和在代码/实验中的位置。

从课堂到项目的标准流程

步骤操作要点
1. 定义任务先说清输入、输出、假设与评价指标，避免只罗列算法名。
2. 建立表示选择逻辑符号、状态空间、特征向量、张量或 MDP 状态。
3. 选择模型根据数据规模、结构先验和计算预算选择规则、搜索、线性模型、神经网络或 RL 方法。
4. 训练/推理明确损失函数、优化过程、推理步骤和失败模式。
5. 评估迭代用与任务匹配的指标验证效果，并记录可复现实验设置。

代码/伪代码示例

下面的代码只保留课堂最需要抓住的骨架；真正练习时再对照 notebooks、讲义和项目文件补全细节。

import torch

X = torch.randn(128, 3)
y = X @ torch.tensor([2.0, -1.0, 0.5]) + 0.3
model = torch.nn.Linear(3, 1)
opt = torch.optim.SGD(model.parameters(), lr=0.05)

for _ in range(200):
    pred = model(X).squeeze()
    loss = torch.mean((pred - y) ** 2)
    opt.zero_grad(); loss.backward(); opt.step()

常见误区

  • 线性模型不等于简单：特征设计、正则化和评估决定实际效果。
  • 测试集不能参与调参，否则性能估计会偏乐观。
考试与汇报提醒：回答问题时尽量使用“任务定义 → 方法原理 → 关键公式/代码 → 指标 → 局限”的结构。
