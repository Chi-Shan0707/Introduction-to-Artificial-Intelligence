现实数据输入输入高维特征矩阵 X ∈ R^{N×d} 和一维标签向量 Y ∈ R^N。进行均值归一化，防止量纲差异。失败于特征多重共线性。
线性映射构建构建前向预测公式 z = Xw + b。若为回归，预测值为 z；若为分类，将 z 输入 Sigmoid 转换为概率。失败于非线性分布无法拟合。
损失目标度量定义优化目标。线性回归使用 MSE (均方误差)；Logistic 回归使用 Cross Entropy (交叉熵损失)。失败于异常值大幅拉偏 MSE。
梯度计算推导对参数 w 和 b 求偏导。使用解析公式计算批量梯度或随机梯度。失败于梯度消失（Sigmoid 饱和区）或梯度爆炸。
参数迭代更新利用梯度下降更新参数：w <- w - η * ∇w。动态衰减学习率 η。失败于步长过大产生震荡不收敛。
收敛与泛化评估计算训练集和测试集上的损失与精度，绘制学习曲线，检测是否过拟合或欠拟合。失败于过拟合测试集性能暴跌。

公式推导：从目标到结论

下面不是孤立公式，而是推导链：先写目标，再说明约束，最后解释算法为什么长成现在这样。

线性回归：ŷ=wᵀx+b；MSE=1/n Σ(ŷᵢ-yᵢ)²。
二分类：p=σ(wᵀx+b)=1/(1+e^{-(wᵀx+b)})。
梯度下降：θ_{t+1}=θ_t-η∇θL(θ_t)。
  1. 从经验风险最小化出发，把训练集误差写成所有样本损失的平均。
  2. 对参数求梯度，梯度方向是损失上升最快方向，负梯度用于下降。
  3. 学习率 η 控制每一步走多远；过大会震荡，过小会收敛慢。
  4. 正则项把模型复杂度写进目标函数，降低过拟合风险。

\n# -*- coding: utf-8 -*-
# 第 4 讲: 线性模型 - NumPy 实现 Logistic 回归梯度下降与交叉熵损失的完整实现
import numpy as np

# 1. 构造数据集 (线性可分数据)
np.random.seed(42)
num_samples = 100
X = np.random.randn(num_samples, 2)
# 真实的超平面 w_true = [1.5, -2.0], b_true = 0.5
y_true = (X[:, 0] * 1.5 - X[:, 1] * 2.0 + 0.5 > 0).astype(int)

# 2. 模型定义
w = np.zeros(2)
b = 0.0

def sigmoid(z):
    return 1 / (1 + np.exp(-np.clip(z, -15, 15)))

# 3. 训练循环 (梯度下降)
lr = 0.1
epochs = 200

for epoch in range(epochs):
    # 前向预测
    z = np.dot(X, w) + b
    p = sigmoid(z)
    
    # 计算交叉熵损失 Loss = -1/N * sum( y*log(p) + (1-y)*log(1-p) )
    loss = -np.mean(y_true * np.log(p + 1e-15) + (1 - y_true) * np.log(1 - p + 1e-15))
    
    # 梯度计算: dL/dw = 1/N * X^T * (p - y), dL/db = 1/N * sum(p - y)
    dw = np.dot(X.T, (p - y_true)) / num_samples
    db = np.sum(p - y_true) / num_samples
    
    # 参数更新
    w -= lr * dw
    b -= lr * db
    
    if epoch % 40 == 0:
        accuracy = np.mean((p >= 0.5).astype(int) == y_true)
        print(f"Epoch {epoch:03d} | Loss: {loss:.4f} | Accuracy: {accuracy*100:.1f}%")

print("\n--- 最终优化参数 ---")
print(f"估计 w: {w}, 估计 b: {b}")
print(f"真实 w_true: [1.5, -2.0], b_true: 0.5")

# 第4讲：损失函数、梯度下降与线性模型
## 本讲提要
- 优化方法：梯度下降、SGD、Mini-batch 与常见改进
- 线性模型：线性回归、逻辑回归、Hinge、感知机
## 损失函数
损失函数（Loss Function）是机器学习里非常核心的概念：
- 模型在做预测后，损失函数用一个数值衡量“预测得有多差”
- 优化器（如梯度下降）沿着“让损失变小”的方向更新参数
## 1. 损失函数、目标函数、代价函数的关系
- 数据集平均损失（经验风险）：所有样本损失的平均
\text{Objective} = \frac{1}{N}\sum_{i=1}^{N} \ell\left(y_i, \hat y_i\right) + \lambda \Omega(\theta)
- $\Omega(\theta)$ 是正则项（控制模型复杂度）
## 2. 回归任务常见损失函数
### 2.1 MSE（均方误差）
\ell_{\text{MSE}} = \frac{1}{N}\sum_{i=1}^{N}(y_i - \hat y_i)^2
- 真实房价（万元）：$[100, 120, 150]$
- 预测房价（万元）：$[110, 115, 170]$
- MSE：$\frac{100 + 25 + 400}{3} = 175$
可以看到第三个样本误差 20，被平方后变成 400，对整体影响特别大。
### 2.2 RMSE（均方根误差）
\ell_{\text{RMSE}} = \sqrt{\frac{1}{N}\sum_{i=1}^{N}(y_i - \hat y_i)^2}
### 2.3 MAE（平均绝对误差）
\ell_{\text{MAE}} = \frac{1}{N}\sum_{i=1}^{N}|y_i - \hat y_i|
- MAE：$\frac{10 + 5 + 20}{3} = 11.67$
### 2.4 Huber Loss（平滑 L1）
\frac{1}{2}e^2, & |e|\le \delta \\
\delta(|e| - \frac{1}{2}\delta), & |e|>\delta
### 2.5 Log-Cosh Loss
\ell = \sum_i \log\left(\cosh(\hat y_i - y_i)\right)
## 3. 二分类常见损失函数
二分类：例如苹果 vs 香蕉（apple vs banana）。
- $\hat p = P(y=1|x)$ 是模型预测为正类的概率
### 3.1 0-1 Loss（理论常见，实践少直接优化）
### 3.2 BCE / Log Loss（二元交叉熵，最经典）
\ell_{\text{BCE}} = -\left[y\log(\hat p) + (1-y)\log(1-\hat p)\right]
- 模型给 $\hat p=0.9$（很自信且正确）
- 损失：$-\log(0.9) \approx 0.105$
- 模型给 $\hat p=0.1$（很自信但错）
- 损失：$-\log(0.1) \approx 2.303$
说明：错误且自信时，损失暴涨，这能强烈推动模型修正。
### 3.3 Hinge Loss（SVM 经典）
先把标签写成 $y\in\{-1,+1\}$，模型输出实数分数 $f(x)$。
\ell_{\text{hinge}} = \max(0, 1 - y f(x))
- 若 $y=+1$，$f(x)=2$，则损失 $\max(0,1-2)=0$（分对且间隔够）
- 若 $y=+1$，$f(x)=0.2$，则损失 $0.8$（分对但不够自信）
### 3.4 Focal Loss（处理类别不平衡）
\ell_{\text{focal}} = -\alpha(1-p_t)^\gamma\log(p_t)
- 若样本分类正确且 $p_t$ 高，$(1-p_t)^\gamma$ 很小，损失被压低
## 4. 多分类常见损失函数
### 4.1 Softmax Cross-Entropy（多分类交叉熵）
模型输出 logits：$z_1,\dots,z_K$，Softmax 概率：
\hat p_k = \frac{e^{z_k}}{\sum_j e^{z_j}}
### 4.2 Label Smoothing Cross-Entropy
思想：不把真实标签设为绝对 1/0，而是“稍微平滑”。
- 平滑后：真实类 $1-\varepsilon$，其它类分到 $\varepsilon$
- 大规模分类模型、Transformer 训练中很常见
## 5. 概率分布匹配类损失
### 5.1 KL Divergence（KL 散度）
D_{KL}(P\|Q)=\sum_x P(x)\log\frac{P(x)}{Q(x)}
- 用分布 $Q$ 近似真实分布 $P$ 时的信息损失
- 非对称：$D_{KL}(P\|Q) \ne D_{KL}(Q\|P)$
### 5.2 JS Divergence（Jensen-Shannon）
基于 KL 的对称版本，更稳定，常见于 GAN 理论分析。
## 6. 排序、检索和度量学习常见损失
### 6.1 Contrastive Loss
### 6.2 Triplet Loss
三元组：anchor、positive、negative。
\ell = \max\{0, d(a,p)-d(a,n)+m\}
## 7. 序列任务常见损失
### 7.1 CTC Loss
适合输入输出对齐未知的序列任务，如语音识别、OCR。
### 7.2 序列交叉熵（Teacher Forcing）
在机器翻译、文本生成中，逐时间步计算交叉熵并求和或平均。

## 13. 梯度下降（Gradient Descent）循序渐进讲解
如果说损失函数定义了“错多少要罚多少”，那梯度下降就是“怎样系统地把罚分降下来”。
### 13.1 先明确目标：我们到底在最小化什么
J(\theta) = \frac{1}{N}\sum_{i=1}^{N}\ell\big(y_i,\hat y_i(\theta)\big)
\theta^* = \arg\min_{\theta} J(\theta)
### 13.2 为什么“导数/梯度”能告诉我们怎么走
一元函数里，导数 $f'(x)$ 描述函数在 $x$ 点的变化趋势：
\nabla J(\theta)=\left[\frac{\partial J}{\partial \theta_1},\frac{\partial J}{\partial \theta_2},\dots,\frac{\partial J}{\partial \theta_d}\right]^\top
- 所以取负梯度方向 $-\nabla J(\theta)$，就是下降最快方向（局部意义）
### 13.3 梯度下降的核心更新公式
第 $t$ 轮参数记为 $\theta_t$，则更新为：
\theta_{t+1}=\theta_t-\eta\nabla J(\theta_t)

w_2=0.6-0.1\cdot(-4.8)=1.08

线性模型深度补充：预测函数—损失函数—梯度下降

线性模型是后续神经网络的基本语法：输入经过线性打分，损失函数衡量预测与目标的差距，梯度下降更新参数。即使 CNN 和 Transformer 更复杂，底层仍反复使用线性层、损失函数、正则项和梯度优化。

ŷ = wᵀx + b
MSE = (1/n)Σᵢ(ŷᵢ-yᵢ)²
θ ← θ - η∇θL

Logistic 回归虽然名字里有“回归”，但在课程语境中通常是线性分类模型。它把线性打分 wᵀx+b 通过 sigmoid 或 softmax 转成概率，再用交叉熵优化。相比 0-1 损失，交叉熵可导且能表达“错得有多自信”。

问题模型输出常用损失解释
线性回归连续值MSE惩罚数值偏差，异常值影响较大
二分类正类概率Binary Cross Entropy惩罚概率分布与标签不一致
多分类类别概率分布Cross Entropy鼓励真实类别概率最大

易错点：线性模型不等于“没有表达力”。特征设计、正则化、数据划分和评价指标会显著改变实际效果；测试集一旦参与调参，性能估计就会偏乐观。
